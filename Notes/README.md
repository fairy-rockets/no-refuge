# テーマ

## 斑鳩 x 友情百合

斑鳩を軸にした２人の交流。それぞれの悩みをうちあけ、仲を深め、成長していく。

## 人生で何をなすべきか、何を為すか、どう為すか

その決断と、努力の物語。

ゲームをクリアしても、誰も、ほめてなどくれない。ゲームには意味はない。そもそも、ただの娯楽、暇つぶしである。

ゲームは人が作ったもの。学校の試験とも似ているけれど、学校での勉強と違って、「大事」なものではない。忘れたっていい。しなくたっていい。むしろ、そんなことしてないで、勉強しなさい、と大人たちはいう。

もちろん、社会の本当の「戦争」や「問題」などでもない。現実のそれらと違って、ゲームは絶対にクリアできる、「勝てる」ことが保障されている。そうでなければ、それはバグだ。

それでも、あなたはやる？血眼になって、そのための努力をする？何十回も、何百回も、…何千回も、同じステージに挑み、同じように死ぬ覚悟は、ある？

そして、斑鳩は、「クリア」できれば、自爆してしまう。敵を本当に倒せたのかも、「平和」になったのかも、主人公の望んだ「自由」を取り戻せたのかも、よくわからない。

…

…なぜ？どうして？

どうして、どこから、そんな「努力」や「覚悟」が生まれてくるの？

それは、ゲームに魅力があるから？斑鳩が、奇麗なグラフィックだから？

それとも…。

そして考える。人生の目的とは何か。人に与えられた目標ではなく、自分の人生を生きるとは何なのか。そんな物語を、「斑鳩」を軸につむげないだろうか。

## 仮想空間で繰り返される、生と死の物語

現実では死んだらそれきり。でも、ゲームは違う。しかも、斑鳩はランダム性がほとんどない。例えて言えば同じ人生を、何度でも繰り返し生きることができる。その中で、あなたは何を思い、何を感じ、何をする？

## 学校とは違う、ゲームセンターだけでの関係

いまどきは珍しいが、昔は珍しくはなかった。それを、現代でよみがえらせる。ゲームセンターが、完全に、無くなってしまう前に。

## テーマはあくまで内面の葛藤

斑鳩の強いボスを倒す過程を物語の主題にはおかない。もちろん、臨場感を出すためにパターンの話はしてもいいし、その発明は斑鳩の醍醐味だ。

主題はあくまで、森羅の言葉で言うならば「なぜ、俺は戦っている？・・・・・自由に生きる為だろう？・・・だが、俺はどこへ行こうとしている？・・・」

だから、先輩に倒し方のパターンを教えてもらうかどうかで三日悩んでもいい。

もちろん、スコア争いの話にもしない。スコア争いをするとすれば、それはあくまで、ゆいやつばめの中での話。残機を増やしクリアのためにスコア稼ぐのか、スコアそのものを目標とするのか。そこが主題。

## 「野球」をテーマにするように、「斑鳩」がテーマの物語があったっていいじゃない！

これは何もいうまい。
